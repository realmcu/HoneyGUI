蜂窝菜单 (Menu cellular)
========================
蜂窝菜单控件以蜂窝形状显示一组图标。

用法
-----

创建蜂窝菜单控件
~~~~~~~~~~~~~~~~~
使用 :cpp:any:`gui_menu_cellular_create` 函数创建蜂窝菜单控件，用户需要提供代表图标集的图像文件数组。

添加点击回调事件
~~~~~~~~~~~~~~~~
使用 :cpp:any:`gui_menu_cellular_on_click` 函数添加点击图标的回调事件。

设置偏移
~~~~~~~~
使用 :cpp:any:`gui_menu_cellular_offset` 函数可以调整水平和垂直偏移以获得更好的显示效果。
