   dc,显示画布

   dx,触摸板沿 x 轴的偏移

   fs,文件系统
   hw,硬件
   id,索引
   img,图像
   info,信息
   init,初始化
   mem,内存
   mq,消息队列
   nz,平面 Z 方向的法向量
   obj,对象
   off,偏移
   pic,图片
   pos,位置
   prev,先前的
   rst,结果
   src,源
   sx,x 方向缩放
   tmp,临时的
