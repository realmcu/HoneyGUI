====
工具
====

在 :term:`UI` 设计中，需要利用资源转换工具将图片、字体、3D 模型、视频等资源转换成二进制文件，并将所有 UI 资源文件进行打包，最后使用烧录工具进行烧录。

本章分为两大部分：

一、资源处理工具

- VSCode 插件工具：集成在 HoneyGUI Visual Designer 中，可视化界面，拖拽操作，自动转换和打包。 
- 独立转换工具：独立 exe 程序，包括图像转换工具、字体转换工具。
- 打包工具：使用脚本打包生成的资源。


二、烧录工具

用于将打包后的资源烧录到设备 Flash 中。
