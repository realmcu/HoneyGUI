============
资源处理工具
============

资源处理工具用于将各种格式的资源文件（图片、字体、3D 模型、视频等）转换为嵌入式设备可用的二进制格式，并进行打包。

**VSCode 插件资源转换工具**

集成在 HoneyGUI Visual Designer 插件中的一体化解决方案：

- **功能**：资源转换 + 自动打包
- **特点**：可视化界面，拖拽操作，一键完成转换和打包
- **输出**：资源 bin 文件 + ``app_romfs.bin`` + ``app_romfs.c`` + ``ui_resource.h``，同时支持文件系统访问和直接内存地址访问
- **适用场景**：多种资源文件、复杂格式转换

**独立转换工具**

需要单独下载和运行的 exe 程序：

- **功能**：仅资源转换
- **特点**：GUI 界面，批量转换
- **输出**：资源 bin 文件
- **适用场景**：独立使用或批量转换
- **注意**：转换后需要使用打包工具打包

**打包工具**

打包脚本依赖于 Python 环境：

- **功能**：打包单独的资源 bin 文件
- **输出**： ``root_xxx.bin`` + ``ui_resource.h``
- **适用场景**：配合独立转换工具使用


.. note::
   推荐使用 VSCode 插件工具，它集成了转换和打包功能，操作更简单，无需额外的打包步骤。
