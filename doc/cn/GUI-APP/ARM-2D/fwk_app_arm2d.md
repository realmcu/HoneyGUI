# 使用ARM-2D设计应用程序

Arm-2D 是一个在 Cortex-M 处理器上进行 2.5D 图像处理的开源项目。

- 初始目标：物联网终端设备、白色家电、手持设备和可穿戴设备，特别是对于资源受限和低功耗需求的设备。
- 初始重点：图形用户界面 (GUI) 开发。
