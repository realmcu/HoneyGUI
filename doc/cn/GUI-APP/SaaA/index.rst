使用可视化工具设计应用程序
===========================

.. note::
   **推荐使用新版本**：我们推荐使用 :doc:`VSCode 插件 <../VSCode/index>` 进行应用程序开发。
   VSCode 插件提供了更现代化的开发体验,包括完全离线支持、代码保护区机制、协同开发等功能。

   本章节介绍的 RVisualDesigner 是传统的桌面工具,适用于已有项目的维护。

请参考： `使用可视化工具设计应用程序 <https://docs.realmcu.com/gui/cn/v1.0.12.0.rvd/GUI-APP/SaaA/index.html>`_。
